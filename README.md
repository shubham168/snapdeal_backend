# snapdeal_backend

This is a repo created for backend used in the snapdeal_clone project.
**Uses express and mongoose**

api routes

- /men
- /men/:id
- /jewellery
- /jewellery/:id
- /bagsFootwear
- /bagsFootwear/:id
- /beautyProducts
- /beautyProducts/:id
- /electronics
- /electronics/:id
- /home_kitchen
- /home_kitchen/:id
- /kids
- /kids/:id
- /women_western
- /women_western/:id
- /women_ethnic
- /women_ethnic/:id

## Each of the above route will have 2 parts a normal get all products and a get by id

Features:

- get all data for different types of products
- get data for a single product by id
- Uses atlas database
